مقدمة

في هذا المقال، سنشرح كيفية تشغيل ChatGPT على جهاز الكمبيوتر الخاص بك بدون الحاجة إلى اتصال بالإنترنت باستخدام أداة Ollama. ستتعرف على كيفية إعداد البيئة المطلوبة واستخدام الأوامر اللازمة لتشغيل الذكاء الاصطناعي محليًا.

الأكواد المستخدمة في الشرح

فيما يلي الأكواد والأوامر المستخدمة في الشرح. يمكنك نسخها واستخدامها في بيئتك الخاصة:

تسطيب واجهة المستخدم

اذا كان لديك كارت شاشة خارجي استخدم هذا الامر :

docker run -d -p 3000:8080 --gpus=all -v ollama:/root/.ollama -v open-webui:/app/backend/data --name open-webui --restart always ghcr.io/open-webui/open-webui:ollama

و اذا لم يكن لديك كارت شاشة استخدم هذا الامر :

docker run -d -p 3000:8080 -v ollama:/root/.ollama -v open-webui:/app/backend/data --name open-webui --restart always ghcr.io/open-webui/open-webui:ollama
